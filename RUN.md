## מסלול 1 — בדיקה מהירה עם שרת MOCK (התחל מכאן! בלי Whisper)

זה מוכיח שכל הצינור (לכידה → offscreen → UI) עובד, בלי להתעסק בהתקנת מודל.

### א. הפעל את שרת ה-mock
ב-PowerShell:
```powershell
cd C:\Users\itayl\OneDrive\Desktop\SUB-PRO\backend
.\.venv\Scripts\python.exe mock_server.py
```
אמור להופיע: `[mock] STT server on ws://localhost:9090`. **השאר את החלון פתוח.**

### ב. טען את התוסף ב-Chrome
1. פתח `chrome://extensions`
2. הפעל **Developer mode** (פינה ימנית-עליונה)
3. **Load unpacked** → בחר את התיקייה:
   `C:\Users\itayl\OneDrive\Desktop\SUB-PRO\extension`
4. ודא שהתוסף נטען בלי שגיאות (אם יש "Errors" אדום — פתח אותם ושלח לי).

### ג. פתח דף עם וידאו בעברית
הכי קל לבדיקה: סרטון YouTube עם דיבור בעברית (הוספתי את youtube.com לתוסף בשביל זה).
לחלופין: דף Moodle עם הרצאה.

### ד. הפעל כתוביות — **דרך אייקון התוסף (מומלץ)**
לחץ על **אייקון התוסף בסרגל הכלים** → בפופאפ לחץ **"▶ הפעל כתוביות"**.
> למה דרך האייקון ולא הכפתור על הסרטון? קליק על האייקון מעניק לתוסף הרשאת `activeTab`,
> שהיא אמינה יותר ל-`tabCapture`. הכפתור הצף על הסרטון אמור לעבוד גם, אבל אם הוא נכשל
> בפעם הראשונה — זו הסיבה; פשוט השתמש בפופאפ.

### ה. מה אמור לקרות
- כתוביות עברית (מזויפות: "שלום לכולם וברוכים הבאים להרצאה") רצות בתחתית הסרטון, מתעדכנות (אפור→לבן).
- **אתה ממשיך לשמוע את הסרטון.**
- לחיצה נוספת (פופאפ "■ עצור") → הכתוביות נעצרות.

אם זה עבד — **הצינור שלם.** אפשר לעבור למסלול 2 (תמלול אמיתי).

## מסלול 2 — תמלול עברי אמיתי (faster-whisper)

### ⚠️ קודם: גרסת Python
ל-faster-whisper צריך **Python 3.11 או 3.12** (ל-3.14 שמותקן אצלך אין עדיין wheels).
התקן Python 3.12 ובנה venv נפרד אם צריך. פרטים: [backend/README.md](backend/README.md).

### א. התקן והרץ את השרת האמיתי
```powershell
cd C:\Users\itayl\OneDrive\Desktop\SUB-PRO\backend
# (בתוך venv של Python 3.11/3.12)
pip install -r requirements.txt
python server.py
```
בהרצה ראשונה יורד מודל `small` (~כמה מאות MB). ברירת מחדל = CPU.
המתן ל-`[stt] ws://localhost:9090 ready`.

### ב. אותם צעדים ב-Chrome (ב-ג-ד-ה ממסלול 1)
עכשיו הכתוביות יהיו **תמלול אמיתי** של מה שנאמר בסרטון.

> איכות נמוכה/איטית על CPU? זה צפוי. לאיכות מיטבית — GPU + מודל עברי מלא (ראה [backend/README.md](backend/README.md), משתני `MHS_*`).

## 🔧 פתרון תקלות

| תסמין | סיבה סבירה / פתרון |
|-------|---------------------|
| הכפתור על הסרטון לא מופיע | הדף לא תואם / הסרטון ב-iframe. השתמש בפופאפ (אייקון). |
| לחצתי הפעל אבל אין כתוביות | השרת לא רץ / endpoint שגוי. בדוק `chrome://extensions` → "service worker" (console) לשגיאות WebSocket. ודא `ws://localhost:9090` ב-⚙️ הגדרות. |
| "Extension has not been invoked" | הפעל דרך **אייקון התוסף** (פופאפ), לא הכפתור על הסרטון. |
| האודיו נעלם כשלוחצים הפעל | לא אמור לקרות (מחברים ל-destination). אם כן — שלח לי, זה באג בחיבור האודיו. |
| אזהרת הרשאת לכידת מסך/אודיו | צפוי ל-`tabCapture`. אשר. |
| השרת לא עולה (mock) | ודא `pip install websockets` בתוך ה-venv. |
| server.py קורס בהתקנה | כנראה Python 3.14 — עבור ל-3.11/3.12. |

### איך לפתוח את ה-console של ה-service worker / offscreen
`chrome://extensions` → התוסף → **service worker** (קישור כחול) פותח DevTools של ה-SW.
ל-offscreen: באותו עמוד תחת "Inspect views" אם מופיע. שם רואים את הלוגים של `[offscreen]`.
